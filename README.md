## PS Hiring Assignment

### Available Scripts

-   Run app in dev mode with CSR

    `yarn start`

-  Build App

    `yarn build`
   
-  Start app with SSR 

    `yarn server`



### Tech Stack

-   React 16 
-   Redux
-   Express
-   Boostrap 4


### Deployment
    
This application is deployed on heroku at following url:
